# 🌍 Multilingual Text Translator

A fully functional Flask-based multilingual text translation app that converts user input into various languages using the Google Translate API.

---

## ✨ Features

- Translate text between 20+ languages
- Clean, modern UI using HTML, CSS, and Flask
- Powered by Googletrans (Google Translate API wrapper)

---
